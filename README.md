# Telegram Automation Assistant 

## Overview
Telegram Automation Assistant is an AI-powered workflow that automates Telegram interactions using automation tools.

The assistant can:
- Process Telegram messages
- Convert voice messages
- Send automated responses
- Integrate with AI agents

## Technologies Used
- n8n
- Telegram Bot API
- AI Agent
- HTTP Requests
- Automation workflows

## Features
- Telegram message automation
- Voice message processing
- AI response generation
- Workflow automation

## Workflow
Telegram Message → Processing → AI Agent → Response sent to Telegram

## Setup
1. Create a Telegram bot using BotFather
2. Import the workflow into n8n
3. Add your Telegram credentials
4. Run the automation
